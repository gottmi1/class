type(타입)

ECMAscript 명세에 따르면,
'타입'이란 자바스크립트 엔진 개발자 모두에게 어떤 값을 다른 값과 분별할 수 있는, 고유한 내부 특성의 집합이다.

    built-in type(내장타입)

  1. null
  2. undefined
  3. boolean
  4. number
  5. stiring
  6. object
  7. symbol(ES6 버전부터)

null

mdn 명세

어떠한 값 자리가 존재는 하나 '값이 비어있음' 을 뜻한다. 정의되지 않는 undefined와 개념을 구분한다.

undefined

mdn 명세

단어 그대로 '정의되어있지 않다.' 라는 뜻이며, 일반적으로 값이 할당되지 않을때 해당 타입을 확인 할 수 있다.

git : 버전관리 시스템 git GUI: 집에 설치해놓으면 작업하는거 자동업로드도 가능
github : 코드공유 , 혹은 협업용 서비스
hosting : 웹 퍼블리싱 주요 단계
respository : 저장소(remote, local)
branch : 또 다른 버전 master,main branch : 메인버전 코드
commit : 파일 업로드할 때 붙여주는 메모
push : 코드업로딩(배열 push와 같음)
pull : 코드 다운로드
issue : 문제점 발견했을 때 표시
pull request : 어떤 코드를 제안할 수 있음(제안서)
